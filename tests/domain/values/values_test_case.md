# valuesでのテスト

## 値Objectを厳格に検証する
正常系： 値Objectが正しく生成されることを確認する。▶制約を満たしている場合をテストする。
異常系： 値Objectが不正な値で生成されないことを確認▶ **厳格にテストする** 

## テストフォーマット
```python
# coding: utf-8
# $$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$
# 仕様：
# 検証の順序 > 型チェック > Noneチェック > 値チェック
# $$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$
# import
import pytest

from src.domain.values import XXX

# ----------------------------------------------------------------------------------


def test_XXX_success():
    ...

# ----------------------------------------------------------------------------------

@pytest.mark.parametrize(
    "invalid",
    [ ... ],
)

def test_XXX_invalid(invalid):
    with pytest.raises(ValueError):
        ...

# **********************************************************************************
```

### @pytest.mark.parametrize(...) の使い方
- 異常系テストで複数のパターンを一括で検証
- 正常系でも複数の値を一括で検証
